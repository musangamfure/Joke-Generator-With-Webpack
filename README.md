# 📖 [Joke Generator App] <a name="about-project"></a>

This is a very simple Joke generator app I developed when I was learning Webpack. if you want to use this environment,replace with your own code and install other packages as needed

## 🛠 Built With <a name="built-with"></a>

### Tech Stack <a name="tech-stack"></a>

```
- HTML5
- SCSS
- Javascript
- Babel Loader
- Sass Loader With Dart Sass
- HTMLWebpackPlugin
- WebpackDevServer
- Asset Resource Loader
- Source Maps
- Caching
```

### Prerequisites

In order to run this project you need:

> Install Git in your computer Git
> Any text editor VSCode
> Install linters on your local env't

### Setup

Clone this repository to your desired folder:

git clone https://github.com/musagamfure/webpack.git

### Usage

Inside your IDE:

Run Dev Server (Port 3000)

```
  npm run dev
```

## Build for production
